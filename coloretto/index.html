  • If the "add" stack from above is full, click again to get a new stack.
  • If the "add" stack from above is "none", take from the stack listed above.
  • If "none" is selected when all stacks are empty, ignore it and click again to get a new "add" stack.
  • If all stacks are full, take from the stack above.
  • If the stack it tries to take from is empty, keep clicking until it receives a valid stack to take from.
The game is set-up as the standard 2-player variant, i.e. remove 2 colors so there are only 5 in the deck.
The automata scores positive points for all stacks. It never scores negative points.
If the automata picks up any jokers, add them to the color which will give the automata the greatest increase in points.
